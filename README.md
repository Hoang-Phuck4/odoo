# 🚀 Odoo 17 HR & Payroll Custom Modules

Dự án này cung cấp các module tùy chỉnh cho hệ thống **Odoo 17**, tập trung vào quản lý nhân sự (HR), chấm công, tính lương và bảo mật hệ thống.

---

## 📌 Tính năng chính

* 👨‍💼 Quản lý nhân sự (Employee Management)
* ⏱️ Chấm công (Attendance)
* 📅 Quản lý nghỉ phép (Leave)
* 💰 Tính lương (Payroll)
* 🔐 Bảo mật mật khẩu người dùng

---

## 🧩 Các module

| Module                    | Mô tả                       |
| ------------------------- | --------------------------- |
| `forher_hr_employee`      | Quản lý thông tin nhân viên |
| `forher_attendance`       | Chấm công                   |
| `forher_leave`            | Quản lý nghỉ phép           |
| `forher_contract`         | Hợp đồng lao động           |
| `forher_payroll`          | Tính lương                  |
| `forher_company_overview` | Báo cáo tổng quan           |
| `password_security`       | Bảo mật mật khẩu            |
